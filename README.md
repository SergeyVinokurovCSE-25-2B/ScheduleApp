# Расписание кафедры

Творческая работа студентов группы ИВТ-25-2б Винокурова Сергея и Марениной Екатерины.

---

## О проекте

Приложение для просмотра расписания кафедры, разработанное на C++ с использованием фреймворка Qt. Программа загружает расписание из JSON-файла и отображает его в удобной табличной форме с группировкой дней недели.

---

## Решаемые проблемы

Студенты и преподаватели сталкиваются с несколькими типичными проблемами при работе с расписанием:

- **Неудобный поиск** — расписание часто разбросано по фотографиям в мессенджерах, бумажным листам или закрытым документам.
- **Путаница с неделями** — чётная и нечётная недели отличаются, легко перепутать.
- **Отсутствие единого интерфейса** — приходится использовать несколько источников для получения информации.

---

## Реализованные функции

- **Просмотр расписания студентов** — выбор группы и чётности недели
- **Просмотр расписания преподавателей** — выбор преподавателя из списка
- **Группировка дней недели** — день показывается один раз, объединяя несколько пар
- **Автоматическая подстройка таблицы** — ширина колонок и высота строк подбираются под содержимое
- **Загрузка данных из JSON** — расписание можно обновить, просто заменив файл
- **Обработка ошибок** — понятные сообщения вместо вылетов программы

## Архитектура

Проект содержит следующие основные компоненты:

- **Структуры данных** (Lesson, DaySchedule, WeekSchedule, TeacherData, GroupData) — отвечают за хранение расписания.
- **ScheduleLoader** — загружает JSON-файл и заполняет структуры данных.
- **MainWindow** — главное окно для выбора роли (студент/преподаватель) и чётности недели.
- **TeacherScheduleWindow** — окно отображения расписания преподавателя.
- **StudentScheduleWindow** — окно отображения расписания студента.

Связь между классами построена по принципу разделения ответственности: загрузка, хранение, навигация и отображение разделены.
